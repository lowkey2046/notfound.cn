使用 spacemacs 和 hugo 写博客
2018-12-23 10:18 label emacs   hugo  

Spacemacs org layer 提供了对 Hugo 的支持。

工作过程：ox-hugo 把 org 文档转换为 Hugo markdown，Hugo 再将 markdown 转换为 html。

Hugo

安装

sudo snap install hugo
sudo apt install python-pygments

搭建

使用 Hyde 主题

hugo new site notfound.cn
cd notfound.cn
git init
git add .
git submodule add https://github.com/spf13/hyde.git themes/hyde

配置

  • 语法高亮 Configure Syntax Highlighter
PygmentsCodeFences = true # Enable syntax highlighting with GitHub flavoured code fences
PygmentsUseClasses = true # Use CSS classes to format highlighted code

编译

hugo -s notfound.cn -d /var/www/notfound.cn
  • -s 指定源目录
  • -d 指定目标目录

本地实时预览：

hugo server -D

远程编译：

ssh -T notfound.cn hugo -s work/notfound.cn -d /var/www/notfound.cn
  • 需要配置 ssh key
  • 直接在服务上执行命令，省掉了登录的步骤

Emacs hugo 配置

修改 spacemacs 配置，开启 org hugo 支持 Hugo support

(setq-default dotspacemacs-configuration-layers '(
  (org :variables
       org-enable-hugo-support t)))

Org 文件配置

ox-hugo 支持两种导出方式。

子节点树作为博客内容（推荐）

将一个子节点树作为一篇博客，可参考 One post per Org subtree (preferred)

博客文件目录结构：

.
├── content
├── content-org
│   └── posts.org
...
  • 所有博客内容都保存在 posts.org 文件中

在 posts.org 头部添加：

#+HUGO_BASE_DIR: ../
#+HUGO_BASESECTION: ./
  • #+HUGO_BASE_DIR: Hugo 博客根目录设置为上级目录
  • #+HUGO_BASESECTION: Hugo section

在 posts.org 子节点添加：

:PROPERTIES:
:EXPORT_FILE_NAME: ubuntu-emacs-and-hugo
:END:

#+HUGO: more
  • EXPORT_FILE_NAME 指定导出的文件名
  • HUGO: more 文件预览截断点，可参考 Hugo Summary Splitter
  • 生成的文件会保存到 HUGO_BASE_DIR/content/posts/ubuntu-emacs-and-hugo.md

ox-hugo 导出时会使用 org 文件的部分信息，可参考 For subtree-based exports

  • org tags 会转换为 Hugo markdown 的 tags
  • CLOSED 的时间点会转换为 Hugo markdown 的 date

Org 文件作为博客内容

可参考 One post per Org file

保存后自动导出 Auto-export on Saving

  • 目录下所有文件导出。在目录下创建文件 .dir-locals.el 并添加如下内容， content-org 目录下的文件都会被自动导出为 hugo markdown：

    (("content-org/"
  . ((org-mode . ((eval . (org-hugo-auto-export-mode)))))))

  • 单个文件自动导出

    ​* Footnotes
​* COMMENT Local Variables                          :ARCHIVE:
# Local Variables:
# eval: (org-hugo-auto-export-mode)
# End:

  • 可配置单个文件不导出

    ​* Footnotes
​* COMMENT Local Variables                          :ARCHIVE:
# Local Variables:
# eval: (org-hugo-auto-export-mode -1)
# End:
